# Healthcare Claim Denial Risk Prediction

This project demonstrates an end-to-end machine learning pipeline for predicting healthcare insurance claim denials using structured claims and billing data.

## Key Features
- Data cleaning and exploratory analysis
- Feature engineering and ML pipelines using scikit-learn
- Batch-based claim risk scoring
- Model explainability and evaluation
- FastAPI service for on-demand scoring
- Production-style scheduling and automation

## Tech Stack
- Python, SQL
- Scikit-learn, Pandas, NumPy
- MLflow
- Power BI (for reporting)
- FastAPI
